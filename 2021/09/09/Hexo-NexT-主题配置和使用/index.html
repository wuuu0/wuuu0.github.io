Hexo NexT 主题配置和使用

发表于 2021-09-09 更新于 2021-12-21

在此前的博客——个人博客搭建，我们成功利用 Hexo 生成了静态网页，使用了NexT博客主题，并将其部署到了wuchaojie.top上。但我们还不知道如何使网页的布局和页面达到我们想要的效果。

归根结底，网页是由 HTML + CSS + Javascript 组成的，它们分别决定了网页的 内容/结构、样式/布局、行为。可想而知，Hexo + 自定义主题（如此次的NexT）就是帮助我们来更加简单地来创造一个网页，让我们能 “简单地” 设置 Hexo和NexT 提供的配置文件 blog/_config.yaml blog/themes/next/_config.yaml 来创建网页以达到我们想要的效果（它们已经实现了大部分效果，我们只能有限地挑选和调整罢了），而不需要去手敲和组织前端三大件（ H C J ）的代码，进而让我们更加专注于网页内容的创作。

简单的网页配置

详细的配置方法见 Hexo官方文档-配置。

在hexo博客工作目录根目录中，打开_config.yaml。

本网站仅对以下内容进行了设置：

参数 描述
title 网站标题 （Wu’s Blog）
subtitle 网站副标题 （CS Tech）
author 您的名字 （wuchoajie）
language 网站使用的语言 （zh-CN）
theme 使用的主题名 （next）

NexT 主题配置

详细的配置方法见 NexT官方文档 主题设置 和 NexT 主题个性化教程。

本网站仅参考NexT 的简单配置在 menu处新增了首页、标签、分类 三个子页面导航，在侧边栏提添加了头像, 。（个人偏好简约风格，好看且可偷懒）
